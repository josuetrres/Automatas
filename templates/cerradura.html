Cerradura Inteligente

Modela el sistema de seguridad de una cerradura digital. Acepta una secuencia de intentos correctos (c) y fallidos (f). Tras 3 fallos consecutivos el sistema se bloquea.

Alfabeto (Σ)

Σ = { c, f }

  • c Intento correcto
  • f Intento fallido
