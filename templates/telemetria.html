AFND-ε – Autómata Finito No Determinista con transiciones épsilon

Validador Telemetría IoT

Valida paquetes de telemetría IoT. Un paquete válido comienza con la cabecera (r), puede incluir lecturas de temperatura (t) y/o humedad (h), y termina con el CRC (c).

Alfabeto (Σ)

Σ = { r, t, h, c }

  • r HDR – cabecera del paquete
  • t Temperatura – dato de sensor
  • h Humedad – dato de sensor
  • c CRC – checksum de validación
